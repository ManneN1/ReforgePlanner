# ReforgePlanner

A reforging optimizer for World of Warcraft: Cataclysm.

ReforgePlanner finds the best reforges for your current gear and can compare all valid combination of candidate items while optimizing reforges for each gear setup combination.

## Features

- Gear reforge optimization
- Candidate item comparison (with reforge optimization)
- Combination Rules
- Wowhead Gear Planner import
- JSON import/export

## Running

Clone the repository and open `index.html` in a browser, or serve the directory with any static web server.
